# Café Yamu — Website

One-page landing site for **Café Yamu** (Köln-Nippes): Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Assets

Images are in `public/images/`:

- `cup.png` — hero to-go cup
- `essen.png` — breakfast / food
- `logo.jpg` — YAMU logo (navbar, gallery)
- `outside.png` — storefront

Replace or add `og.jpg` in `public/images/` if you want a dedicated Open Graph image.
